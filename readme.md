# scam.py
A python implementation of [scam.m][scam], a  matlab symbolic circuit solver script.

## dependencies:
  - [python3][py3] (no plan for py2 support)
  - [sympy][sympy] (should be as simply as "`pip install sympy`")

## Usage:
```shell
scam.py <netlist-file-path>
```
